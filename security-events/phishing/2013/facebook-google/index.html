# 事件详情

在 2013 年至 2015 年期间，Facebook 和谷歌成为虚假发票诈骗的受害者，损失超过 1 亿美元。

在该起骗局中，立陶宛黑客 Evaldas Rimasauskas 成立了一家冒牌公司，冒充台湾电脑制造商 广达电脑（Quanta Computer），并与 Facebook 和谷歌建立合作关系。

攻击者在包括 塞浦路斯和拉脱维亚 在内的几个国家中，开设了与假冒公司同名的银行账户用于洗钱。随后，Evaldas 继续向 Facebook 和谷歌的员工发送虚假发票，引导他们将所需的资金电汇给他。

然而，他最终被逮捕，被正式指控犯有电汇欺诈罪，并强制没收 4970 万美元。
