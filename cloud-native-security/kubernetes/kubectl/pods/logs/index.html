kubectl logs

# 查看 Pod 命令行输出

kubectl logs <Pod名称>
1

# -c选项

查看 Pod 中指定容器的命令行输出（当 Pod 中具有多个容器时，默认对第一个容器进行操作）。

kubectl logs <Pod名称> -c <容器名称>
1
