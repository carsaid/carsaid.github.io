AWS(亚马逊云)

# AWS(亚马逊云)

# AWS 相关链接

AWS 官网：https://aws.amazon.com/ (opens new window)

AWS 中文官网：https://aws.amazon.com/cn/console/ (opens new window)

AWS 文档：https://docs.aws.amazon.com/ (opens new window)

AWS 中文文档：https://docs.amazonaws.cn/ (opens new window)

# AWS 云主机 - EC2

# 名称

AWS 的云主机名称为EC2

# AWS 存储桶 - S3

# 名称

AWS 的存储桶名称为S3

也可以称为S3 Bucket

# 存储桶托管 - 静态网站

在存储桶中，用户可以托管一个或多个静态网站，类似于 github.io 。

每个托管在 S3存储桶 中的网站，都会被分配一个 AWS 的域名，你不需要购买额外的域名，也不用配置 DNS 。

# 存储域名的命名规则

<bucketname>.s3-website-<region>.amazonaws.com

  • bucketname（存储桶名称）
  • region（地区）

# 存储端点的命名规则

s3-<region>.amazonaws.com/<bucketname>

  • bucketname（存储桶名称）
  • region（地区）

# 存储示例

  • 你创建了一个名为abc.com的 S3存储桶 ；
  • 存储桶位于美西部2区；
  • AWS 会自动为其分配一个域名：abc.com.s3-website-us-west-2.amazonaws.com；
  • 存储端点为：s3-us-west-2.amazonaws.com/abc.com。
