# DNSLOG平台ceye.io支持

ceye.io是一个DNSLOG平台，注册即可获得专属的域名，用于辅助无回显漏洞的验证，官网：http://ceye.io/ (opens new window)

vulcat默认使用的DNSLOG平台是http://www.dnslog.cn/ (opens new window)

如果你想使用ceye.io，则需要在官网进行注册，在你的个人主页获得专属的 域名和Token

然后将你的 域名和Token 填写到config.yaml对应的地方，保存文件即可
