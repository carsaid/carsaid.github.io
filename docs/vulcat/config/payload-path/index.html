# 配置攻击载荷路径

vulcat默认的攻击载荷路径为./payloads/

在进行漏洞检测时，vulcat将会自动搜索该目录下的有效载荷，并对检测目标使用

你可以：

  • 编写自己的载荷 并放到该路径下
  • 或 更改载荷路径
