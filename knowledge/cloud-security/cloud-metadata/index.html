各大公有云厂商-元数据服务地址

# 各大公有云厂商-元数据服务地址

  • AWS（亚马逊云）：http://169.254.169.254/latest/meta-data/
  • Google Cloud（谷歌云）：http://metadata/computeMetadata/v1/
  • Azure（微软云）：http://169.254.169.254/metadata/instance?api-version=2017-04-02
  • Oracle Cloud（甲骨文云）：http://169.254.169.254/opc/v1/instance/
  • 阿里云：http://100.100.100.200/latest/user-data/
  • 腾讯云：http://metadata.tencentyun.com/latest/meta-data/

# 亚马逊云 - 详情

地址 作用
http://169.254.169.254/latest/meta-data/iam/info 获取当前服务器的 IAM 信息
http://169.254.169.254/latest/metadata/iam/security-credentials/<rolename> 获取指定角色的 IAM 临时凭据。其中<rolename>需要替换为相应的角色名称。
