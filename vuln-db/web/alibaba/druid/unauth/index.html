Alibaba Druid未授权访问

# 💛 Alibaba Druid-未授权访问

# 1）漏洞信息

漏洞名称 受影响组件 漏洞类型 漏洞编号
阿里巴巴Druid未授权访问 Alibaba Druid (opens new window) 未授权访问 暂无
漏洞简介

Druid是阿里巴巴数据库事业部出品，为监控而生的数据库连接池。

Druid提供监控功能，监控SQL的执行时间、Web URI的请求、Session等。

当开发者配置不当时就可能造成未授权访问漏洞。
